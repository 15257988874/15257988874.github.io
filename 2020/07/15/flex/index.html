flex 什么

flex 是 flexible box 的缩写,意味弹性布局,用来为盒状模型提供最大的灵活性。任何一个容器都可以被设定为 flex(设定后 float,clear,veitical-align 失效)

基本概念

采用 flex 布局的元素,称之为 flex 容器,他的所有子元素自动成为容器成员(flex 项目)

容器默认存在两根轴：水平的主轴（main axis）和垂直的交叉轴（cross axis）。主轴的开始位置（与边框的交叉点）叫做 main start，结束位置叫做 main end；交叉轴的开始位置叫做 cross start，结束位置叫做 cross end。

项目默认沿主轴排列。单个项目占据的主轴空间叫做 main size，占据的交叉轴空间叫做 cross size。

容器的属性

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

1.flex-direction
主轴的方向,值可能为 row | row-reverse | column | colunmn-reverse

  • row(默认值):水平,起点在左边
  • row-reverse:水平,起点在右边
  • column:垂直,起点在上端
  • colunmn-reverse:垂直,起点在下端

2.flex-wrap
项目如何换行,值可能为 nowrap | wrap | wrap-reverse

  • nowrap(默认值):不换行
  • wrap:换行
  • wrap-reverse:换行,第一行在下方

3.flex-flow
flex-dirction 和 flex-wrap 的缩写,默认值为 row nowrap

4.justify-content
项目在主轴上的对其方式,值可能为 flex-start | flex-end | space-between | space-around | center

  • flex-start(默认值):左对齐
  • flex-end:右对齐
  • center:居中
  • space-between:两端对齐,项目之间的间隔相等
  • space-around:两端对齐,每个项目两侧的间距相等(项目之间的间距比项目与边框的间距大一倍)

5.align-items
项目在交叉轴上的对其方式,值可能为 flex-start | flex-end | center | baseline | stretch

  • flex-start:交叉轴的起点对齐
  • flex-end:交叉轴的终点对齐
  • center:交叉轴的中点对齐
  • baseline:项目的第一行文字的基线对齐
  • stretch(默认值):如果项目未设置高度或设为 auto，将占满整个容器的高度

6.align-content
项目多根轴线时候的对齐方式(一根轴此属性无效),值可能为 flex-start | flex-end | space-between | space-around | stretch | center

  • flex-start(默认值):交叉轴的起点对齐
  • flex-end:交叉轴的终点对齐
  • center:交叉轴的中点对齐
  • space-between:与交叉轴两端对齐，轴线之间的间隔平均分布
  • space-around:每根轴线两侧的间隔都相等。所以，轴线之间的间隔比轴线与边框的间隔大一倍
  • stretch(默认值):轴线占满整个交叉轴

项目属性

  • order
  • flex-grow
  • flex-shrink
  • flex-basic
  • flex
  • align-self

1.order

项目的排序顺序,数值越小,越靠前

2.flex-grow

项目的放大比例,默认为0,即如果存在剩余空间,也不放大

3.flex-shrink

项目的缩小比例,默认为1,即如果空间不足,该项目就缩小(负值无效,0 不缩小)

4.flex-basic

flex-basic 为 width 或 height 的替代品,定义了在分配多余空间之前，项目占据的主轴空间（main size）。浏览器根据这个属性，计算主轴是否有多余空间。它的默认值为auto，即项目的本来大小。
它可以设为跟 width 或 height 属性一样的值（比如 350px），则项目将占据固定空间。

5.flex

flex-grow,flex-shrink,flex-basic 的缩写,默认值0 1 auto,后 2 个属性可选,该属性有两个快捷值:auto (1 1 auto) 和 none (0 0 auto)。

6.align-self

单个项目有与其他项目不一样的对齐方式，可覆盖 align-items 属性。默认值为 auto，表示继承父元素的 align-items 属性，如果没有父元素，则等同于 stretch。
