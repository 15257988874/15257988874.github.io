JavaScript数组方法速查手册极简版

发表于 2019-12-13 | 分类于 JavaScript | |

前言

JavaScript 数组方法速查手册极简版中共收了 32 个数组的常用方法和属性，并根据方法的用途进行重新排序和分类，在文中简要的介绍了方法作用和用例说明,改变元素组的方法reverse,pop,push,shift,unshift,splice

类型判断

Array.isArray() 数组类型判定。

创建数组

Array.of

从可变数量的参数创建数组，不管参数的数量或类型如何。

Array.from

用类数组或可迭代对象创建新数组。

查找元素

find-按函数查找

找到第一个满足检测函数条件的元素，并返回该元素，没找到则返回 undefined。

findIndex-按函数查找

找到第一个满足检测函数条件的元素，并返回该元素索引。找不到返回-1。

indexOf-按元素值查找

查找元素并返回元素索引值，找不到返回-1
第一个参数代表查找的值,第二个参数表示查找的起始位置。

lastIndexOf-按元素值查找

从后向前查找元素并返回元素索引值，找不到返回 -1。

includes-值包含检测

值包含检测，如包含返回 true，不包含返回 false。

some-函数包含检测

检测数组中是否有元素可以通过检测函数验证,如包含返回 true，不包含返回 false。存在就终止循环

every-函数完全检测

检测数组中是否所有元素都可以通过检测函数验证。如全部符合返回 true，否则 false

添加元素

push-尾部添加

在尾部添加一个或多个元素，返回数组的新长度

unshift-头部添加

在头部添加一个或多个元素，并返回数组的新长度

删除元素

pop-尾部删除

尾部删除一个元素，并返回该元素

shift-头部删除

从头部删除一个元素，并返回该元素

替换元素

添加、替换、删除元素。会改变原数组

array.splice(start,deleteCount,[item1[, item2[, ...]]]])

  • 参数 start：表示替换的位置
  • 参数 deleteCount ：表示删除元素的数量
  • 参数 item1… ： 表示添加的元素

顺序相关

sort-排序

数组排序，改变原数组,支持自定义规则排序

reverse-反序

倒置数组，并返回新数组。会改变原数组

遍历迭代

forEach

forEach 按升序为数组中含有效值的每一项执行一次 callback 函数，那些已删除或者未初始化的项将被跳过（例如在稀疏数组上）,遍历数组中的元素，并执行回调函数，返回 undefined
注:无法中断(只能抛出异常)
arr.forEach(callback(currentValue , index array),thisArg );

  • currentValue：数组中正在处理的当前元素。
  • index(可选) ：表示删除元素的数量
  • array(可选) ： 表示添加的元素
  • thisArg(可选) ： 当执行回调函数 callback 时，用作 this 的值
